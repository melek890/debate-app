# debate-app  

A web-based debate platform that allows users to participate in structured debates with assigned positions, timed rounds, and audience voting.

## Features
- Create or join debate rooms
- Assigned Pro / Con positions
- Timed debate rounds
- Audience voting
- Debate history

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB

## Goal
This project was built to promote critical thinking, structured argumentation, and respectful discussion among students.
